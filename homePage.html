GET /idol
gets you information on all idols
GET /idol/:name
provides information on a specific idol
POST /idol
requires an ID, Name, and Unit. Sub_Unit optional. adds a new idol to the table
PATCH /idol/:name
requires a Name and Unit. Sub_Unit optional. lets you modify an idol's information
DELETE /idol/:name
deletes an idol from the table
